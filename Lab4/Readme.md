About the Dataset

The dataset used in this assignment is a Walmart sales dataset, which contains weekly sales data for multiple stores.
It includes columns such as Store, Date, Weekly_Sales, Holiday_Flag, Temperature, Fuel_Price, CPI, and Unemployment.
Among these, Weekly_Sales, Temperature, Fuel_Price, CPI, and Unemployment are numerical features that represent the store performance, economic conditions, and weather factors affecting sales. 
The dataset is suitable for data preprocessing and analysis because it contains a mixture of numerical and categorical features, potential missing values, and possible outliers.

What Was Done

In this assignment, the dataset was preprocessed by assessing data quality, handling missing values with the mean, 
removing outliers using the IQR method, normalizing numerical features with Min-Max and Z-score scaling, and applying PCA to reduce dimensionality when strong correlations existed. 
These steps ensured the data was clean, consistent, and ready for analysis.
