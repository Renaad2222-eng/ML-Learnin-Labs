Dataset Description
This project uses the Walmart Store Sales Dataset, which contains historical weekly sales data for different Walmart stores. 
The dataset includes store information, weekly sales figures, holiday indicators, temperature, fuel prices, consumer price index (CPI), and unemployment rates.
The data covers the period from February 2010 to November 2012 and allows for time-based
analysis and evaluation of business performance under different economic conditions.

Assignment Description
This assignment aims to explore and understand the Walmart sales dataset using Exploratory Data Analysis (EDA). 
The analysis includes cleaning the data, checking for missing values, examining patterns and trends, and looking at the effect of holidays on sales. 
The goal is to uncover insights about weekly and monthly sales, identify key patterns, and better understand the factors influencing store performance.
