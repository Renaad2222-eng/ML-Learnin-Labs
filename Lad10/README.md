what was done
In this lab, we analyzed the Iris dataset to classify flower species into three categories: setosa, versicolor, and virginica.

We started with exploratory data analysis to understand the relationships between features such as sepal length and sepal width using visualizations like pairplots and KDE plots.

Then, we prepared the data by splitting it into training and testing sets to ensure proper model evaluation.

After that, we built a Support Vector Machine (SVM) classifier to predict the species of the flowers. We evaluated the model using a confusion matrix and classification report to measure its performance.

Finally, we improved the model using GridSearchCV to tune hyperparameters (C and gamma), which resulted in better accuracy and overall classification performance.
